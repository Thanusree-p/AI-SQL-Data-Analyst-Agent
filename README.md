# AI SQL Data Analyst Agent

## Overview
This project converts natural language queries into SQL queries and generates insights with visualization.

## Features
- CSV to SQL database
- Natural Language → SQL
- Data analysis
- Visualization using Plotly

## How to Run
pip install -r requirements.txt
streamlit run app.py

## Note
Add your GROQ API key in .env file:
GROQ_API_KEY=your_key_here
